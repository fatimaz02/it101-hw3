Extra Credit — What to Do & Avoid

Home (Q1) Q2

Five Things You SHOULD Do

  • Use semantic HTML tags.
  • Include descriptive alt text for images.
  • Use consistent fonts and spacing.
  • Make pages responsive.
  • Keep CSS separate from HTML.

Five Things to AVOID

  • Walls of text with no headings.
  • Auto-playing sounds or videos.
  • Too many different fonts/colors.
  • Vague link text like “click here”.
  • Using images for text content.
